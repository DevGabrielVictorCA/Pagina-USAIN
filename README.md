# USAIN — Website

🌐 Acesse: https://usainsolucoes.com.br/

## 📌 Descrição

Este repositório contém o código-fonte do website institucional da USAIN.

O projeto foi desenvolvido utilizando HTML, CSS e JavaScript puro, com foco em performance, responsividade e clareza na apresentação das informações.

---

## 🚀 Funcionalidades

- Estrutura institucional completa  
- Seção de serviços  
- Seção de depoimentos  
- Formulário de contato  
- Navegação responsiva  
- Animações de scroll com AOS  
- Layout adaptado para diferentes dispositivos  

---

## 🛠️ Tecnologias utilizadas

- HTML5  
- CSS3  
- JavaScript  
- AOS (Animate On Scroll Library)

## ⚙️ Como executar o projeto

Não é necessário instalar dependências.

1. Clone o repositório:

```bash
git clone https://github.com/DevGabrielVictorCA/Pagina-USAIN.git
```

2. Acesse a pasta do projeto:

```bash
cd Pagina-USAIN
```

3. Abra o arquivo `index.html` no navegador
